## 🚀 Como Executar

### Pré-requisitos
- Node.js 18+ 
- npm ou yarn

### Instalação
```bash
# Clone o repositório
git clone https://github.com/samirpegado/sg-portfolio

# Entre no diretório
cd sg-portfolio

# Instale as dependências
npm install

# Execute em modo de desenvolvimento
npm run dev
```

### Build para Produção
```bash
# Gere o build otimizado
npm run build

# Execute o build
npm start
```
